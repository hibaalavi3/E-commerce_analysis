# E-commerce Analysis with Python
## Overview
This project conducts an in-depth analysis of social media engagement metrics for an e-commerce platform using Python. The analysis aims to optimize content strategies and enhance audience engagement based on insights derived from the dataset.
## Key Findings
- Engagement Insights: 80% of posts received less than 500 likes, indicating moderate engagement levels.
- Post Type Dominance: Photos dominated the content landscape, representing 40% of total posts, followed by videos at 30%.
- Audience Demographics: The majority of the audience, comprising 50%, fell within the 25-34 age bracket, with females accounting for 60%.
- Category Performance: The food category emerged as the most popular, constituting 25% of total posts, while fitness-related posts demonstrated the highest average likes per post at 500.
- Sentiment Analysis: Likes comprised 40% of reactions, with 25% expressing love, indicating a predominantly positive sentiment towards the content
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
## File Structure
- data/: Contains the dataset used for analysis.
- notebooks/: Includes Jupyter notebooks for data preprocessing, exploratory data analysis (EDA), and visualization.
- scripts/: Contains Python scripts for data cleaning and preprocessing.
- README.md: Overview of the project and key findings.
## How to Use
- Clone the repository to your local machine.
- Navigate to the notebooks directory.
- Open the Jupyter notebooks to explore the analysis process and findings.
- Run the scripts in the scripts directory for data cleaning and preprocessing if necessary.
- Review the README.md file for a summary of the project and key insights.
## Results
The analysis provided actionable insights for optimizing content strategies, fostering deeper audience engagement, and driving business growth on social media platforms.
